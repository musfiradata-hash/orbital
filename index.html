About this dashboard — it now looks for real pipeline output at data/ehi_live.json on load. If found, matching zones are marked ● LIVE and use real Earth Engine numbers; any zone without a match, or if the file isn't found at all, falls back to the illustrative synthetic data so the demo never breaks. Important: most browsers block this kind of file fetch when you open the HTML file directly (double-click / file://) — it needs to be served over HTTP, e.g. via GitHub Pages, or locally with python -m http.server run from the folder containing both this file and data/ehi_live.json.
